# Firmware Notes

This directory is the entry point for implementation-level documentation.

- `legacy-firmware-features.md`: old firmware behavior and user-visible compatibility notes.
- `new-firmware-dev-notes.md`: current firmware modules, call chains, and implementation notes.
- `boot-ota.md`: bootloader and OTA implementation notes, if enabled.
- `app-changelog.md`: application firmware release notes.
- `boot-changelog.md`: bootloader release notes.
