# Hardware Notes

This directory is optional.

Use it for hardware explanations that help firmware work, such as power sequencing, boot pins, USB topology, storage wiring, connector pinout, or board-level constraints.

Required hardware files should live under the repository `hardware/` directory instead:

```text
hardware/schematic/     Schematic PDF/source, recommended
hardware/pcb/           PCB source or exports, optional
hardware/production/    Gerber, BOM, placement files, optional
```
